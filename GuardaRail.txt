# LiqPagoEstandar
Aplicación que le genere un resumen de pago mensual estándar para el personal de casa de familias de clientes de un estudio contable

## Stack
- Angular 17.3.17 + SQL Server 2022
- API en C# para consultas y escritura a la Base de Datos

## Cómo correr
 - Tests: `pytest -q`
